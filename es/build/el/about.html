Πώς ξεκίνησε

Η ιδέα για το Togather γεννήθηκε στα πρώτα στάδια του αποκλεισμού COVID-19 στο Ηνωμένο Βασίλειο. Είδαμε ένα άρθρο σχετικά με τις ολλανδικές ειδήσεις όπου μια οικογένεια κράτησε ένα λεύκωμα για τον συγγενή τους στη Μονάδα Εντατικής Θεραπείας (ICU) και της το έδωσαν όταν απολύθηκε από το νοσοκομείο. Έγραψαν τι συνέβη σε αυτήν ενώ ήταν στο ICU, αλλά τι συνέβη και στον έξω κόσμο, ειδήσεις, πράγματα που είχε κάνει η οικογένεια. Νομίζαμε ότι αυτό ήταν πολύ ωραίο και θέλαμε να το κάνουμε διαθέσιμο για άτομα που περνούν παρόμοιες εμπειρίες.

Μέσα σε αυτές τις δύσκολες στιγμές όπου οι οικογένειες και οι φίλοι χωρίζονται, μπορεί να νιώσουμε αδύναμοι. Ελπίζουμε ότι μέσω της δημιουργίας μιας ομάδας και την ενασχόληση με διάφορα θέματα και δραστηριότητες, παρέχουμε ένα χώρο για να αντλήσετε δύναμη και να νιώσετε ελπίδα. Ένας χώρος όπου παρόλο που η καθημερινότητα συνεχίζει να είναι περίεργη, μπορείτε να δημιουργήστε κοινές αναμνήσεις μαζί με το αγαπημένο σας πρόσωπο στο κέντρο όλων αυτών. Θέλουμε να σας βοηθήσουμε να υποστηρίξετε την οικογένεια / τους φίλους / την κοινότητά σας να φτιάξει και να μοιραστεί ιστορίες, όταν η συνεύρεση δεν είναι δυνατή.

Το απόρρητό σας

Με απλά λόγια. Έχουμε σχεδιάσει το Togather για να διατηρήσουμε τα μηνύματα και τα μέσα σας ιδιωτικά. Όλα τα μηνύματα συνομιλίας και τα αρχεία πολυμέσων σας παραμένουν στη συσκευή σας και δεν κοινοποιούνται ούτε αποθηκεύονται ποτέ από το Togather (ή οποιοδήποτε άλλο μέσο).

Τα μόνα δεδομένα που συλλέγουμε είναι μέσω της ανώνυμης ανάλυσης ιστότοπων και μέσω της προαιρετικής φόρμας σχολίων στο τέλος της συναρμολόγησης του βιβλιαρίου σας. Είμαστε μια ομάδα ερευνητών (παρακάτω λεπτομέρειες), που θα θέλαμε να ακούσουμε για τις εμπειρίες σας με το ToGather (καθώς και τυχόν προτάσεις για το πώς να το βελτιώσετε). Αυτό είναι απολύτως εθελοντικό και σε καμία περίπτωση δεν επηρεάζει την πρόσβασή σας ή τη χρήση του ιστότοπου ToGather. Για να υποστηρίξουμε αυτά τα σχόλια και να λάβουμε κάποια εικόνα σχετικά με τον τρόπο με τον οποίο οι χρήστες χρησιμοποιούν το ToGather, καταγράφουμε ποια θέματα κοινοποιούνται - αυτό είναι. Δεν καταγράφουμε δεδομένα χρήστη ή πληροφορίες που μπορούν να εντοπιστούν σε κανέναν. Για να σας δώσουμε μια ιδέα για τα δεδομένα που συλλέγουμε έχουμε δημιουργήσει ένα παράδειγμα ημερολογίου για μια μέρα, κάντε κλικ εδώ για να τα δείτε.

Είμαστε μέρος του ερευνητικού προγράμματος που ονομάζεται Enabling Ongoingness (www.enablingongoingness.com) το οποίο διερευνά πώς μπορούμε να δώσουμε νέο νόημα στις σχέσεις στο πλαίσιο του τέλους της ζωής και του πένθους. Ένα μεγάλο μέρος αυτού διερευνά πώς οι άνθρωποι επιμελούν την ουσία μιας σχέσης στα μέσα μαζικής ενημέρωσης, ιστορίες, αντικείμενα και τελετές στην καθημερινή ζωή. Κατά τη διάρκεια της πανδημίας COVID-19 αντιμετωπίσαμε μια νέα παγκόσμια κοινή εμπειρία φόβου για απώλεια και θλίψη. Σε αντίθεση με οτιδήποτε έχουμε βιώσει πολλοί από εμάς και έχει αλλάξει αναμφίβολα τη ζωή μας με πολλούς τρόπους. Μέσω του ToGather ελπίζουμε ότι μπορούμε να σας βοηθήσουμε να συλλάβετε τους κοινοτικούς δεσμούς σας σε μια εποχή όπου αυτό δεν είναι τόσο εύκολο όσο πριν. Εάν έχετε οποιεσδήποτε εμπειρίες με το ToGather που θέλετε να μοιραστείτε μαζί μας, μην διστάσετε να επικοινωνήσετε μαζί μας. Μπορείτε είτε να μας στείλετε ένα μήνυμα μέσω e-mail (kyle.montague@northumbria.ac.uk) ή να συμπληρώσετε τη φόρμα σχολίων.

Togather

Το Togather είναι μια διαδικτυακή υπηρεσία που στοχεύει στην προσέγγιση οικογενειών και φίλων μέσω της ανταλλαγής ιστοριών για ένα αγαπημένο άτομο. Είναι κατασκευασμένο με τις υπάρχουσες υπηρεσίες WhatsApp και GitHub, για να είναι ευρέως διαθέσιμες και δωρεάν για όλους. Θα σας παρέχουμε καθημερινά θέματα και δραστηριότητες για την οικογένειά σας και τους φίλους σας για να στείλετε μήνυμα και να μετατρέψετε τις ομαδικές σας συνομιλίες σε βιβλίο ή λεύκωμα για δώρο, διατήρηση και επίσκεψη.
